# **App Name**: Teal Market

## Core Features:

- Secure Google OAuth Login: Implements Google OAuth for secure user authentication, requiring GOOGLE_CLIENT_ID in .env. Implements the secure login using NextAuth
- Phone Listing Browsing: Allows users to browse available mobile phones listed for sale.
- Detailed Product Pages: Provides detailed product pages with specifications and images for each phone.
- Simple 'Buy Now' Button: Implements a straightforward 'Buy Now' button for immediate purchase actions.

## Style Guidelines:

- Primary color: Soft teal (#64CCC5) for a calming and trustworthy feel.
- Background color: Light teal (#E5F4F3) to complement the primary color.
- Accent color: Pale Green (#A3EBB1) for a contrasting and fresh feel. Use this color in the button.
- Body and headline font: 'Inter' sans-serif for a modern, neutral look.
- Use minimalist icons for navigation and key features to maintain a clean aesthetic.
- Maintain a clean and spacious layout with plenty of white space to focus attention on product listings.
- Implement subtle transitions and animations on button interactions and page loading for a smooth user experience.
